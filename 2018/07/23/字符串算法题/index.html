无重复字符的最长子串(面试题48:最长不含重复字符的子字符串)

题目描述

给定一个字符串，找出不含有重复字符的最长子串的长度。

示例：
给定 “abcabcbb” ，没有重复字符的最长子串是 “abc” ，那么长度就是3。
给定 “bbbbb” ，最长的子串就是 “b” ，长度是1。
给定 “pwwkew” ，最长子串是 “wke” ，长度是3。请注意答案必须是一个子串，”pwke” 是 子序列 而不是子串。

算法思路

  • 遍历字符串，每循环至下一个字符，判断该字符是否包含在前面的子串中，若不在则继续遍历，若在子串中则找到重复的位置，从重复的位置后一位重新纪录
  • 动态规划:(dp[i]记录以i结尾的不包含重复字符的子字符串的最大长度)
    1.如果第i个字符之前没出现过，则dp[i] = dp[i-1] + 1
    2.如果第i个字符之前出现过，则要分两种情况讨论
    找到第i个字符上次出现的位置，并记录下i和该位置的距离差值d
    1)若d小于等于dp[i-1]，则该字符在以i结尾的字串中已出现过，故取dp[i] = d
    2)若d大于dp[i-1]，则该字符出现在dp[i-1]对应字串之前，第i个字符没有重复，故dp[i] = dp[i-1] + 1
